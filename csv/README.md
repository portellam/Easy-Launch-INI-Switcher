# Databases
Comma separated values or `.csv`, are database files.

| Database                | Usage                                                            |
| ----------------------- | ---------------------------------------------------------------- |
| [`dashboards.csv`]      | Dashboard types and revisions.                                   |
| [`directory-paths.csv`] | Accepted root directory keywords.                                |
| [`mount-paths.csv`]     | Available root/mount headers.                                    |
| [`permutations.csv`]    | Possible permutations of `launch.ini`.                           |
| [`plugins.csv`]         | Plugin order and maximum accepted amounts.                       |
| [`plugin-paths.csv`]    | Plugin types and the accepted directory keywords.                |
| [`stealth-servers.csv`] | Stealth server availabilities, compatibilities,and requirements. |
